# Constructor.io Autocomplete UI

[Constructor.io Autocomplete UI](https://github.com/Constructor-io/constructorio-ui-autocomplete) is a React library for building autocomplete search experiences powered by Constructor.io. It provides both ready-to-use components and a headless hook for full customization.

## Prerequisites

- Node.js 14+
- React 16.12+
- Constructor.io API key

> **[Get started with the Quickstart guide](?path=/docs/getting-started-quickstart--docs)**

## Components

- [CioAutocomplete](?path=/docs/components-cioautocomplete--docs) - Main autocomplete component
- [AutocompleteResults](?path=/docs/components-autocompleteresults--docs) - Results dropdown container
- [SearchInput](?path=/docs/components-searchinput--docs) - Search input with form
- [SectionItem](?path=/docs/components-sectionitem--docs) - Individual result item
- [SectionItemsList](?path=/docs/components-sectionitemslist--docs) - Section container with items
